Methods

(static) reset(container, button, resetBtn, currentIndex) → {number}

En function som sätter om form container, gömmer återställningsknapp, sätter användaren kan början på nytt.
Parameters:
Name Type Description
container HTMLElement Container elementen där formen visas.
button HTMLElement Button element som adderar ny form i ordning.
resetBtn HTMLElement Återställningsknapp element som återställer spelet.
currentIndex number Det aktuella indexet som spårar framstegen för tillagda former.
Returns:
Returnerar 0 för att återställa aktuellt index för att starta spelet igen.
